How do I write a basic web scraper in Python that extracts all links from a page?

Here's a minimal Python web scraper using requests and BeautifulSoup:

Install dependencies

pip install requests beautifulsoup4

The scraper

import requests from bs4 import BeautifulSoup from urllib.parse import urljoin


def get_links(url: str) -> list[str]:     """Fetch a page and return all absolute links found in it."""     response = requests.get(url, timeout=10)     response.raiseforstatus()


soup = BeautifulSoup(response.text, "html.parser")     links = []


for tag in soup.find_all("a", href=True):         absolute = urljoin(url, tag["href"])         links.append(absolute)


return links


if name == "main":     target = "https://example.com"     found = get_links(target)     for link in found:         print(link)

What it does

  1. requests.get — downloads the HTML of the page.
  2. BeautifulSoup — parses the HTML into a searchable tree.
  3. soup.find_all("a", href=True) — finds every <a> tag that has an href attribute.
  4. urljoin — converts relative paths like /about into full URLs.

Tips

  • Always check a site's robots.txt and terms of service before scraping.
  • Add delays between requests to avoid overloading servers.
  • For JavaScript-heavy pages, consider playwright or selenium instead.

How would I extend this to only keep links pointing to the same domain?

DeepSeek 2025-06-02 14:07

Use urllib.parse.urlparse to compare the netloc (domain) of each link against the starting URL:

from urllib.parse import urljoin, urlparse


def getsamedomain_links(url: str) -> list[str]:     """Return only links that share the same domain as the input URL."""     base_domain = urlparse(url).netloc


response = requests.get(url, timeout=10)     response.raiseforstatus()     soup = BeautifulSoup(response.text, "html.parser")


links = []     for tag in soup.find_all("a", href=True):         absolute = urljoin(url, tag["href"])         if urlparse(absolute).netloc == base_domain:             links.append(absolute)


return list(set(links))  # deduplicate

The key addition is:

base_domain = urlparse(url).netloc           # e.g. "example.com" if urlparse(absolute).netloc == base_domain:  # only keep same-domain links

Wrapping in set() at the end removes any duplicates (the same link appearing multiple times on the page).
