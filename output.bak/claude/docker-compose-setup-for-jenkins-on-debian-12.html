Here's your Jenkins docker-compose.yml file! Before starting Jenkins, you'll need to:

1. Create the directory structure: 

sudo mkdir -p /lump/apps/jenkins/jenkins_home sudo chown -R 1000:1000 /lump/apps/jenkins/jenkins_home

2. Start Jenkins: 

docker-compose up -d

3. Get the initial admin password: 

sudo cat /lump/apps/jenkins/jenkins_home/secrets/initialAdminPassword

Key features:

  • Jenkins web UI accessible at http://your-server-ip:8080
  • Agent port 50000 for connecting build agents
  • Docker socket mounted so Jenkins can run Docker commands
  • Automatic restart unless manually stopped
  • All data persists in /lump/apps/jenkins/jenkins_home

Optional customizations:

  • Uncomment the JENKINS_OPTS line if you want Jenkins at a subpath like /jenkins
  • Update JAVA_OPTS with your actual server IP
  • Adjust port mappings if 8080 is already in use

  • All data persists in /lump/apps/jenkins/
