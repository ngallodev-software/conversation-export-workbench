3. MVP: the smallest thing Microsoft would buy

MVP name (internal)

“AI Agent Directory & Audit Layer”

MVP does ONLY 5 things

  1. Agent Registry

- Create agents - Assign role (e.g. “Support Triage Agent”) - Owner (team / app)

  1. Role-Based Permissions

- Which tools an agent can call - Which APIs, prompts, or datasets - Simple policy file (JSON / YAML)

  1. Execution Logging (Audit First)

- Every request - Inputs / outputs - Tool calls - Model used - Timestamp + request ID

  1. Performance Metrics

- Cost per task - Success / failure - Latency - Retry count

  1. Kill Switch

- Disable agent instantly - Revoke tokens - Freeze execution

🚫 No autonomy 🚫 No planning 🚫 No fancy orchestration

This is governance software, not an agent framework.
