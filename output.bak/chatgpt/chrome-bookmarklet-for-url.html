Sure! You can create a JavaScript bookmarklet that will pass the URL of the current page as a parameter when clicked. Here's how you can create one for Chrome on Android:

Steps:

  1. Open Chrome on your Android device.
  2. Create a new bookmark with any URL (e.g., https://example.com).
  3. Save the bookmark and then edit it.
  4. Replace the URL with the following JavaScript code:

javascript:(function(){window.location.href='https://yourwebsite.com?url='+encodeURIComponent(window.location.href);})();

Explanation:

  • This script redirects you to https://yourwebsite.com, appending the current page's URL as a parameter named url.
  • Replace https://yourwebsite.com with the desired destination URL that will handle the passed parameter.

Example:

If you want to pass the current URL to Google Search, you could use:

javascript:(function(){window.location.href='https://www.google.com/search?q='+encodeURIComponent(window.location.href);})();

Now, when you tap this bookmark while browsing, it will execute the script and pass the current page URL as a parameter.
