Ah, you're looking to run ChemGPT using Ollama? Great choice if you're working on molecular generation, cheminformatics, or drug discovery tasks using large language models. Let’s walk through it step-by-step so you can get ChemGPT running locally with Ollama. 🔬⚗️🧠

✅ Step 1: Install Ollama (if not already)

Download and install Ollama for your OS: 👉 https://ollama.com/download

✅ Step 2: Search or Pull ChemGPT Model

As of now, ChemGPT may not be officially listed in Ollama's model library yet. You can check the latest available models:

  • Trending: https://ollama.com/library?sort=popular
  • Newest: https://ollama.com/library?sort=newest

If ChemGPT isn't listed there, you have two options:

✅ OPTION A: If ChemGPT is available on Ollama (e.g., via ollama run chemgpt)

Try running:

ollama run chemgpt

If it works, great! You’re done 🎉

✅ OPTION B: Build and Run ChemGPT Locally (from GGUF or Model Repo)

If it's not yet officially available, you can manually add a ChemGPT model. Here's how:

🔧 Step-by-Step to Import Custom ChemGPT Model:

  1. Find the GGUF or GPTQ Model:

- Search places like Hugging Face: https://huggingface.co/models - Look for ChemGPT converted to GGUF format or a format compatible with llama.cpp (which Ollama uses under the hood). - Good starting point: https://huggingface.co/benyaminahmed/ChemGPT-2.7B (convert to GGUF if needed)

  1. Prepare a Modelfile:

Here’s an example Modelfile for a ChemGPT model:

FROM llama2:7b


PARAMETER temperature 0.7 PARAMETER top_p 0.9


Replace with the actual model path you’ve converted or downloaded


WEIGHTS ./chemgpt-2.7b.Q4KM.gguf


TEMPLATE """ You are ChemGPT, an expert chemistry assistant. Answer questions about molecules, reactions, and drug discovery in detail.


{{ .Prompt }} """

  1. Build the Model:

ollama create chemgpt -f Modelfile

  1. Run the Model:

ollama run chemgpt

🧪 Test It Out

Try prompts like:

> Predict the SMILES notation for aspirin. > What is the binding affinity of imatinib to BCR-ABL? > Generate a novel small molecule that inhibits EGFR.

💡 Pro Tip

You can fine-tune ChemGPT on your own datasets using Ollama’s upcoming fine-tuning capabilities, or via model adapters if you're using Hugging Face + GGUF exports.
